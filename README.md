# Кроссворды по The Elder Scrolls

Генератор печатных кроссвордов по миру TES: Skyrim, Morrowind, Oblivion + общий лор и мемы серии.

## Запуск

Открыть `index.html` **двойным кликом**. Браузер откроет страницу через `file:///` — без сервера, без интернета. Работает в Edge, Chrome, Firefox.

Для режима **«3D-сцена»** лучше запускать через локальный сервер, потому что браузеры строже относятся к загрузке ES-модулей и GLB/glTF через `file:///`:

```bash
py -3.14 -m http.server 8013 --bind 127.0.0.1
```

Потом открыть `http://127.0.0.1:8013/index.html`. Модели класть в `assets/models/`, список ожидаемых имён есть в `assets/models/README.md`.

## Использование

1. Жми **«Сгенерировать»** — получаешь новый кроссворд (13×13 по умолчанию).
2. Решаешь прямо в браузере (клик в клетку, набирать с клавиатуры) или печатаешь — **`Ctrl+P`** → «Сохранить как PDF».
3. Кнопка **«Показать ответы»** — для проверки.
4. **«Печать сетки»** — только клетки, без вопросов (если хочешь решить ручкой и не подсматривать).

### Два стиля подсказок

- **«Прямые (фактические)»** — обычные определения, фактическое описание слова.
- **«Хитрые (с намёком)»** — для **100 самых знаковых TES-слов** написаны 6 хитрых формулировок (мемов, цитат, образных описаний). При каждой генерации выбирается случайная — повторно решая, увидишь другие. Примеры:
  - **НАЗЕЕМ** → «Единственный норд, которого убивают даже в самом законопослушном прохождении.»
  - **ФУСРОДА** → «Три слога, после которых мебель и враги летят одинаково.»
  - **ЛИДИЯ** → «Верный хускарл, готовая умереть от твоего же файрбола в спину.»
  - **АЛДУИН** → «Тот, чей крик «Зу мии лон» обещает конец времён.»

### Хитрые формулировки

**Для 100 самых знаковых слов** написано по 6 хитрых формулировок — половина в литературно-психологическом стиле, половина в стиле фан-мема с конкретными игровыми отсылками. При каждом размещении на сетке выбирается случайная, при повторной встрече слова — другая. Это даёт «aha-моменты» в кроссворде:
- ЦИЦЕРОН → «Танцующий безумец, чью фразу про "масло" помнят все.»
- ХАДВАР → «Первый друг в Хелгене, чей дядя работает кузнецом в Ривервуде.»
- РАЛОФ → «Брат Бури, чья сестра держит таверну в Ривервуде.»
- КУРИЦА → «Пернатый детектор зла, превращающий всю деревню в линчевателей.»
- МАЙК → «Пушистый торговец, чьи мудрые изречения цитируют чаще, чем Нельсона Манделу.»

## Технические подробности

- **Стек**: vanilla HTML+JS+CSS, без бандлеров, без интернета.
- **3D-дракон**: локальный Three.js + GLTFLoader; без модели показывает процедурный fallback, с GLB/glTF подхватывает настоящий меш.
- **Кроссворды**: классические, размеры 11×11 / 13×13 / 15×15 / 21×21 / 25×25.
- **Темы оформления**: светлая и тёмная.
- **Печать**: A4, всё-в-одном или раздельно (сетка / вопросы).
- **История**: до 200 последних кроссвордов хранятся в localStorage, не повторяются.
- **Свой пак**: «✎ Редактор корпусов» — можешь редактировать или добавить свои слова. Готовый `.js` загружается через «+ Загрузить свой пак».

## Размер словарного запаса для пересечений

Пак содержит много коротких 3-7-буквенных слов (норды, расы, кричи Тху'ум), которые отлично сцепляются в кроссворде. Сетки 13×13 обычно дают 17-22 слова с пересечениями. На 21×21 — 40-50 слов.
