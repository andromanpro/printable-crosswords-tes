# Dragon model drop zone

The cinematic dragon scene looks for local models in this order:

1. `the_elder_scrolls_blades_ancient_dragon.glb`
2. `tes-blades-ancient-dragon.glb`
3. `tes-blades-ancient-dragon/source/model.glb`
4. `tes-blades-ancient-dragon/source/Dragon_Ancient_Skeleton/Dragon_Ancient_Skeleton.fbx`
5. `alduin/source/Ps Alduin Dragon.glb`
6. `tes-blades-ancient-dragon/scene.gltf`
7. `tes-blades-shulkunaak.glb`
8. `tes-blades-shulkunaak/scene.gltf`
9. `skyrim-dragon.glb`
10. `skyrim-dragon/scene.gltf`
11. `alduin.glb`
12. `alduin/scene.gltf`
13. `dragon.glb`
14. `dragon/scene.gltf`

Recommended animated TES-like base:

- The Elder Scrolls Blades Ancient Dragon  
  https://sketchfab.com/3d-models/the-elder-scrolls-blades-ancient-dragon-67441fe2ce9d4d028e5370faaf4c4b14
  Prefer the `GLB Converted format` download. Put the downloaded `.glb` at
  `assets/models/the_elder_scrolls_blades_ancient_dragon.glb` or
  `assets/models/tes-blades-ancient-dragon.glb`. If Sketchfab gives a zip with
  `scene.gltf`, extracting it to `assets/models/tes-blades-ancient-dragon/` is
  also supported.

Fallback TES look:

- The Alduin Dragon  
  https://sketchfab.com/3d-models/the-alduin-dragon-dd04d9599a2d40b49a26d197b12b475c

Downloaded model files are intentionally ignored by git here. Keep attribution/license notes with any model you use or distribute.
